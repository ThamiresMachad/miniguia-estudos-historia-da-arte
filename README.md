# miniguia-estudos-historia-da-arte
Mini guia de estudos sobre a historia da arte utilizando o notebookLM

##Contexto e Objetivos
Este repositório contém meu projeto de aprendizado ativo sobre **História da Arte**, utilizando a IA **NotebookLM** como ferramenta de curadoria e síntese. O objetivo foi entender a transição entre o Renascimento e o Modernismo.

##Engenharia de Prompts

Prompt Inicial
Resume os principais períodos da história da arte com base nos documentos.

Prompt Estratégico
Agindo como um especialista em arte, e me apoiando unicamente nos materiais dados, vou elaborar um quadro que compara o Renascimento e o Barroco, abordando: 1) A forma como a luz é usada, 2) Os temas mais importantes e 3) A organização dos elementos na imagem. Caso a fonte não fale sobre algum desses aspectos, vou escrever 'Informação não disponível

Prompt para Revisão
Descubra os 5 termos especializados que mais aparecem nos textos analisados (por exemplo: Chiaroscuro, Antropocentrismo) e elabore um glossário. As definições devem refletir o entendimento desses termos segundo os autores das obras consultadas.
